## 网站介绍

都市天气网主要提供天气预报15天查询、未来天气预报、热门城市天气、全国天气地图、天气预警、旅游天气、历史天气、卫星云图等信息入口。

相关二级域名用于承载不同方向的内容，包括萍乡天气查询、在线成语词典、汽车信息和油价信息等。
